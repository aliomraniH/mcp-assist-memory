## What it is

A remote MCP (Streamable HTTP) server, Python 3.11, FastAPI + FastMCP, storage
in PostgreSQL + pgvector. It exposes 18 tools at `/mcp` (bearer-authed),
`GET /healthz` (open liveness), a streamed `GET /artifact/{sha256}`, and a
password-gated `/admin` dashboard that shows and rotates the live MCP token.
Every per-project tool takes a `namespace` (the project/tenant). Data is durable
across redeploys because everything lives in Postgres.

2. Secrets (Replit Secrets, never files):
   - `ADMIN_PASSWORD` — REQUIRED to use `/admin`. Generate a strong value

     where to read it; never echo it.
   - `MCP_AUTH_TOKEN` — REQUIRED. Seeds the first MCP token on initial boot

   - `DATABASE_URL` — leave to managed Postgres; do not set by hand.
   - Optional: `SESSION_SECRET`, `MAX_ARTIFACT_BYTES`. `PORT` is provided by
     Replit.

3. Deploy gate (run WITHOUT a database so tests don't touch any real data):
       pip install -e ".[test]"
       pytest          # run with DATABASE_URL UNSET
   Expect "8 passed, 21 skipped" (the 21 are real-Postgres tests, validated
   upstream). If anything FAILS, STOP and report the output verbatim.

4. Deploy (Reserved VM). The `.replit` run command already does the right thing:
       python scripts/migrate.py && uvicorn app:app --host 0.0.0.0 --port ${PORT:-8000}
   Migration is idempotent. If the DB is unreachable at boot the process exits
   on purpose so the VM restarts — that is correct, not a bug to patch.

5. Get the token + verify (replace $URL, and $TOKEN with the value shown in the
   dashboard):
   - Open `https://$URL/admin`, sign in with `ADMIN_PASSWORD`, copy the live token.
       curl -s $URL/healthz                                   # {"status":"ok","db":"ok"}
       curl -s -o /dev/null -w '%{http_code}' -X POST $URL/mcp/   # 401 (anonymous)
       curl -s -X POST "$URL/mcp/" \
         -H "Authorization: Bearer $TOKEN" \
         -H "Content-Type: application/json" \
         -H "Accept: application/json, text/event-stream" \
         -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-03-26","capabilities":{},"clientInfo":{"name":"verify","version":"0"}}}'
       # expect a JSON-RPC result naming server "assist-memory"; NOT a 401
   If any check fails, report the response body and status and stop.

6. Durability proof (the done-gate). Using the token, call tools/call
